## 💻 安裝

### Git 安裝

在博客根目錄裡安裝穩定版【推薦】

```powershell
git clone -b master https://github.com/fineNineJun/hexo-theme-febr.git themes/febr
```

## ⚙ 應用主題

修改hexo配置文件`_config.yml`，把主題改為`febr`

```
theme: febr
```
